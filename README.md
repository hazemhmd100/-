# دفتر المقهى

تطبيق كاشير وحسابات محلي للمقهى. يعمل من المتصفح ويحفظ البيانات على نفس الجهاز.

## التشغيل

```powershell
npm start
```

ثم افتح:

```text
http://localhost:5509/
```

يمكن أيضاً تشغيل `تشغيل-التطبيق.bat`.

## الفحص

```powershell
npm run check
npm test
```

## النسخ الاحتياطي

عند تشغيل التطبيق عبر `server.cjs`، يحاول البرنامج حفظ نسخة JSON داخل مجلد `backups` في نفس مجلد التطبيق، إضافة إلى خيار التنزيل/المشاركة من الواجهة.

احفظ نسخة خارج الجهاز بشكل دوري، مثل Google Drive أو WhatsApp لنفسك.

## ملاحظات اعتماد

- التطبيق مصمم لجهاز واحد.
- لا تستخدمه من عدة أجهزة في نفس الوقت إلا بعد إضافة قاعدة بيانات ومزامنة.
- سجل التدقيق داخل التطبيق يساعد على معرفة آخر عمليات التعديل والحذف والاستيراد.
